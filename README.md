### 功能
1. 右上角增加 按钮，一键隐藏左侧和中间的面板
2. 修改窗口最小宽度为 600px
3. 设置输入框高度最大为 300px
4. 增加自动展开功能，在菜单里点击即可切换
5. 增加面板左右留白功能，提供2种方式，像素和百分比
